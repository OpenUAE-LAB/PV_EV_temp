Data Privacy

The "EV-PV app" requires access to the SMA collected data for solar plants owned by the same company that launches "EV-PV app". This application requires access to read-only data and is not intended any changes or harm to the existing solar units.

This application is intended for internal usage and all its related APIs will be shared privately.

This application only stores user authentication data but does not store any private information about any individul.
